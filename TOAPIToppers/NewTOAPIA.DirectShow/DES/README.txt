DirectShow Editing Services (DES)

The API in this directory comes from the QEdit.dll file in System32.  They represent
the editing services for DirectShow.

This is also imported as the DexterTypeLib

If your application is not performing any editing operations, you're not likely
to need these particular interfaces.

But, there are a couple of classes that are of general interest for those who 
are writing applications that do capturing.

NullRenderer
SampleGrabber
